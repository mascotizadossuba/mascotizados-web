## Publicar rápido
### Opción A — Netlify
1) Entra a Netlify y arrastra la carpeta descomprimida.
2) Te da un link inmediato.
3) Si ya tienes dominio, lo conectas desde Netlify.

### Opción B — GitHub Pages
1) Crea repo y sube TODOS los archivos a la raíz.
2) Settings → Pages → Deploy from branch (main / root).
3) Guarda y espera el link.
